Timestamp Microservice

This API service converts dates and Unix timestamps.

Usage:

Pass a date string or a Unix timestamp (in milliseconds) to the /api/:date_string? endpoint.

If no date string or timestamp is provided, the API will return the current time.

Example Usage:

  • [project url]/api/2015-12-25
  • [project url]/api/1451001600000
  • [project url]/api/ (for current time)

Example Output:

For a valid date string or timestamp:

{"unix":1451001600000, "utc":"Fri, 25 Dec 2015 00:00:00 GMT"}

For an invalid date:

{"error":"Invalid Date"}

For the current time (when no date is provided):

{"unix":1678886400000, "utc":"Wed, 15 Mar 2023 12:00:00 GMT"} (Example values, actual current time will vary)
